## 🤖 系統架構

```
Yahoo Finance 全市場批量粗篩（chunk=150）
  ↓ 量能門檻：成交量 ≥ 1,500 張 + 收盤價 > MA30
FinMind 精確股價＋法人籌碼細部製卡
  ↓ 動作訊號判斷
輸出至 plum_blossom_data.json → index.html 展示
```
